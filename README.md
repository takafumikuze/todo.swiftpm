# TodoApp

SwiftUIで作成したシンプルなiOSアプリです。

## 機能

- 画面中央にOKボタンを配置
- ボタンをタップするとアラートを表示

## iPad Swift Playgroundsでの開き方

1. このリポジトリをiPadのSwift Playgroundsアプリで開く
2. アプリを実行してテストする

## プロジェクト構成

- `Package.swift` - Swift Packageの定義ファイル
- `Sources/MyApp.swift` - アプリのエントリーポイント
- `Sources/ContentView.swift` - メインビュー（OKボタンを含む）
